python

geting started: 
        easy_install basedata
look update data: 
        from basedata import BaseData

        bd = BaseData()
        bd.get({'md5': ''})
if you have an account: 
        from basedata import BaseData

        bd = BaseData()
        bd.login(username, password)
        bd.set({'md5': ''}, {'openlibrary': 'OL...'})

curl

look up data: 
        curl "https://basedata.org/?md5=123"

setup your own server

        git clone https://github.com/basedata/basedataserver
        cd basedataserver
        ./manage.py syncdb
        ./manage.py runserver
and pass the url to your local server to BaseData 
        from basedata import BaseData
        bd = BaseData('http://127.0.0.1:8000/')
